# AuthorEngine
A virtual console¹ that you code in Ruby.

![animated_gif](#)

# Getting Started
## Install
Assuming you have a CRuby installed:

`bundle install`

## Run
`ruby author_engine.rb`
### Options
`--native` Open AuthorEngine in fullscreen and maximized.
